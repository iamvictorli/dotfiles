**Generated:** 2026-01-26

## OVERVIEW

GNU Stow-managed dotfiles for macOS. Stowable packages live under `stow/` and are symlinked into `~`.

## STRUCTURE

```
dotfiles/
├── install              # Bootstrap script (stow + fnm + trash-cli)
├── Brewfile             # Homebrew dependencies
├── benchmark-zsh        # Zsh startup perf testing (hyperfine)
├── task-loop            # OpenCode task loop runner
└── stow/
    ├── claude/          # Claude Code → ~/.claude/
    ├── zsh/             # Shell config → ~/.zshrc
    ├── tmux/            # Multiplexer → ~/.tmux.conf
    ├── nvim/            # Neovim (LazyVim) → ~/.config/nvim/
    ├── ghostty/         # Terminal → ~/.config/ghostty/
    ├── starship/        # Prompt → ~/.config/starship.toml
    ├── lazygit/         # Git TUI → ~/Library/Application Support/lazygit/
    ├── yazi/            # File manager theme → ~/.config/yazi/
    ├── ssh/             # SSH config → ~/.ssh/config
    └── opencode/        # OpenCode → ~/.config/opencode/
```

## WHERE TO LOOK

## CONVENTIONS

- **Theme:** Tokyo Night Storm everywhere (ghostty, tmux, lazygit, yazi, fzf)
- **Font:** JetBrainsMono Nerd Font Mono
- **Editor:** nvim (aliased from `vim`, `c`)
- **Safe delete:** `rm` aliased to `trash` (trash-cli)
- **VCS:** Check for `.jj/` before git commands
- **macOS paths:** `stow/lazygit/` uses `~/Library/` (non-portable)

## ANTI-PATTERNS (THIS PROJECT)

- DO NOT edit `stow/zsh/.cache/*.zsh` — auto-generated by `_cache_init()`
- DO NOT commit `.DS_Store` (should be gitignored)
- DO NOT add `claude` to the install script's stow package list — excluded intentionally
- NEVER use `eval "$(tool init zsh)"` — use cached init pattern

## COMMANDS

```bash
# Install
./install                    # Bootstrap all
brew bundle                  # Install Brewfile deps

# Test shell startup
./benchmark-zsh              # Measures zsh init time

# Stow individual package
stow -v -d stow -t ~ zsh     # Symlink zsh/ to ~
stow -v -d stow -t ~ -D zsh  # Unstow

# OpenCode task loop
./task-loop                  # Run until PRD complete
```
